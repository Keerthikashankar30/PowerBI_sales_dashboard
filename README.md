# 📊 Sales Performance Dashboard | Power BI

## 📌 Overview

The **Sales Performance Dashboard** is an interactive Power BI project developed to provide comprehensive insights into sales, revenue, profit, customer behavior, product performance, and market segments. The dashboard helps businesses monitor key performance indicators and make data-driven decisions through intuitive visualizations and dynamic filtering.

---

## 🎯 Objectives

* Monitor overall sales performance.
* Analyze sales trends over time.
* Identify top customers and best-performing products.
* Evaluate revenue and profit metrics.
* Understand segment-wise business performance.
* Discover factors affecting profitability.
